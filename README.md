# WeMediaChain Wallet-Core

## 安装环境
### 第一步、配置ElectronMirror

> 在`.zshrc`或`.bash_profile`中配置`Eeletron`下载路径

`export ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron/`

### 第二步、安装依赖

`npm install --save-dev`

## 启动

### 开发

`npm run start`
`npm run electron:dev`

### 本地预览

`npm run preview`

## 打包

`npm run release`
